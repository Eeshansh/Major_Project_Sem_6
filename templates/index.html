Important Security Notice

Please read before using the detector.

This application performs static URL analysis using 27 lexical and structural URL features, rule-based heuristics, WHOIS/domain intelligence (where available), and an embedded Random Forest Machine Learning model.

The detector evaluates only the submitted URL. It DOES NOT analyse:

  • Destination webpages after opening the link
  • Redirect chains
  • Downloaded files or malware payloads
  • JavaScript execution
  • Real-time threat intelligence services
  • Website content or visual similarity

The displayed verdict is an advisory risk assessment.

Always verify the authenticity of unknown URLs before opening them or entering any passwords, banking information or personal details.

Best Practices

  • Verify the sender independently.
  • Use official websites whenever possible.
  • Avoid clicking unknown links.
  • Never share credentials on suspicious websites.

⚠️ Scope & Limitations

Scope of Detection

This detector performs static URL analysis using rule-based heuristics, lexical feature engineering and a Random Forest Machine Learning model. The generated verdict should be considered an advisory risk assessment rather than a definitive security guarantee.

  • Does not analyse webpage content after opening the URL.
  • Does not inspect redirected destinations or redirect chains.
  • Does not execute JavaScript or downloaded files.
  • Does not detect malware payloads or exploits.
  • Does not use real-time threat intelligence feeds.
  • Users should always verify unknown URLs before visiting or entering sensitive information.
